### 基色

#### 暗黑模式

| 名称             | 色号                          | 用途                       |
| ---------------- | ----------------------------- | -------------------------- |
| **背景色**       | **#141414**                   | 整体背景（layout、body）   |
| **组件容器背景** | **#1F1F1F**                   | card、modal、dropdown 背景 |
| **浮层背景**     | **#262626**                   | popover、tooltip、menu     |
| **一级文本**     | **rgba(255, 255, 255, 0.85)** | 主文本颜色                 |
| **二级文本**     | **rgba(255, 255, 255, 0.65)** | 次文本颜色                 |
| **三级文本**     | **rgba(255, 255, 255, 0.45)** | 弱文本颜色                 |

#### 明亮模式

| 色号                                | 用途                    |
| ----------------------------------- | ----------------------- |
| **#FAFAFA**                         | 整体背景（亮灰白）      |
| **#F5F5F5**                         | 最常用的内容背景        |
| **#FFFFFF**                         | 组件背景（Card、Modal） |
| **#000000E0**（≈ rgba(0,0,0,0.88)） | 主文本                  |
| **#000000A6**（≈ rgba(0,0,0,0.65)） | 次文本                  |
